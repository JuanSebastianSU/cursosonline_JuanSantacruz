Página de prueba (DOM + API de Cursos)

    Gestión de Cursos (DOM + llamadas a tu API)

    Explora, agrega y elimina cursos. (Pasa el mouse por el título arriba para ver un evento)

      Documentación de la Interactividad

      • Selección por id/class/tag: en codigo.js se usan getElementById, querySelector, getElementsByTagName.
      • Modificar texto: el título muestra el total de cursos; en renderCursos() se arma el innerText de cada ítem.
      • Estilos dinámicos: color del título (mouseover/out), tamaño de fuente en la descripción y classList.toggle() para resaltar/ocultar.
      • Crear/Eliminar DOM: renderCursos() usa createElement/appendChild y el botón elimina el último li con removeChild.
      • Eventos: click (botones), mouseover/mouseout (título), keydown (Enter crea un curso).
      • Navegación por pestañas: delegación de eventos en el menú y carga de parciales HTML con fetch().
